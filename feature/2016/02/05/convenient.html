Pearl Features

About Pearl

Pearl is a portable lyrics presentation device. Small and lightweight, the Pearl and it's accessories fit into any musicians kit bag. The Pearl is elegant, looks well and can be permanently in place at home in the living room, a church meeting room or office.

Pearl uses open source technology including Raspberry Pi and OpenLP. To find out more on how to get the most out of these products visit http://www.raspberrypi.org and http://www.openlp.org. It is highly recommended to read the OpenLP Manual available at http://manual.openlp.org/.

Maintainable

There are always new songs to enjoy and share. Maintaining the database of songs is trouble free. Add new songs directly or import them from a USB flash drive or even from [CCLI Song Select](http://www.ccli.com/). See the [OpenLP Song documentation](https://manual.openlp.org/songs.html) for an extensive list of the import options available. This is all possible because of the broad range of features in OpenLP but does require a keyboard and mouse. The Pearl has 4 USB ports for adding input devices and removable storage. Moreover, Pearl has an optional wireless keyboard touchpad which is just the right size for portability and usability.

Convenient

Automatic

Pearl makes lyrics presentation easy, convenient and hassle free. Pearl automatically starts once plugged in and automatically reloads the last saved set list.

WiFi Hotspot

The device provides a WiFi Hotspot, with a unique SSID so more than one can operate in close proximity, for effortless connectivity. In particular this makes it possible to use any smart phone for selecting songs and controlling what is presented. When the Pearl is plugged into a network through the Ethernet port the device can provide wireless access for others to access the network. This means it is possible to set up a convenient guest wireless network if desired.

Versatile

Portable computer

Pearl is essentially a computer in your pocket. With the Raspberry Pi at the heart of it the device gives you a lot of options to do more. All you need is a screen, power and an input device such as a keyboard & mouse to edit documents, browse the internet, play games and watch videos. For example, the device comes with LibreOffice suite, Claws Mail and Chrome browser installed. The list of possibilities is huge.

Powerful software

Pearl uses the feature rich OpenLP to manage and present song lyrics. OpenLP is free open source Church worship presentation software. Being open source also means that the developers are continuously working to improve this application, and welcome any comments or questions users may have. Although the Pearl primarily uses, and enhances, the OpenLP Songs feature, all the other features such as support for LibreOffice presentations, Bibles, Media, Custom Slides and Stage View are all there. Do check out the OpenLP website to find out how much this amazing software application in your pocket can be put to use.

Easy to use

Select and go:

The Pearl auto starts once plugged in and will load the last saved set list. That means the songs can be selected ahead of time, saved and within minutes of getting to the venue (friend’s sitting room or hotel meeting room) everything is ready to go.

Any smartphone:

Any smartphone with a javascript enabled browser can be used to select songs. The web application interface is so intuitive anyone can run the lyrics presentation from their smartphone. Unlikely other smartphone based lyrics presentation apps, no one has to hand over their own phone to anyone else in order to run it. What’s more, there’s nothing to install if the smartphone already has a javascript enabled web browser!

Connected

Video:

Supplied with a 1.5 metre HDMI cable and a HDMI / VGA connector the Pearl is ready to be used with the majority of televisions and projectors in homes, hotels, churches and convention centres. The audio out port can also be used to for RCA composite Audio Visual output. That means the Pearl has a video connection option for almost every popular display out there.

Wireless Control:

The selection and presentation of songs can be controlled from any smartphone. The smartphone can connect to the Pearl WiFi Access Point. Alternatively, the Pearl can plug in the an existing network through it’s Ethernet port. The wireless keyboard & touchpad is a convenient option to add or import songs on the go. Although any USB or Bluetooth keyboard and mouse can be used.

USB:

The Pearl has 4 USB ports which can be used to import songs from a USB flash drive or plug in an additional input device.

Compact

Only 9cm x 6.5cm x 2.5cm

Small enough to take with you anywhere. As you can see, it easily fits in a child’s hand. Even with the supplied power adapter, compact HDMI cable and HDMI / VGA connector, the Pearl fits in any musician’s gear bag. It takes up less space than a laptop and weighs less too.

Slot slot in almost anywhere

The Pearl can be discretely positioned behind, under, above or beside the TV or projector. That means it can be secured in place to. The Pearl is controlled wirelessly so no direct access is required once it is powered on.
